### statusup
```
*statusup <stat>{,<char_id>};
```

This command will change a specified stat of the invoking character up by one
permanently. Stats are to be given as number, but you can use these constants to
replace them:

* bStr -  Strength
* bVit -  Vitality
* bInt -  Intelligence
* bAgi -  Agility
* bDex -  Dexterity
* bLuk -  Luck
