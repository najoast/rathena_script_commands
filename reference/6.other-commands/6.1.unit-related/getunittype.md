### getunittype
```
*getunittype <GID>;
```

Returns the type of object from the given Game ID. Returns -1 if the given GID does not
exist.

Return values:
* BL_PC   - Character object
* BL_MOB  - Monster object
* BL_PET  - Pet object
* BL_HOM  - Homunculus object
* BL_MER  - Mercenary object
* BL_NPC  - NPC object
* BL_ELEM - Elemental object
