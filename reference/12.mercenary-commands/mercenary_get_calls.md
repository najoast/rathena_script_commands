### mercenary_get_calls
```
*mercenary_get_calls(<guild>);
```
### mercenary_set_calls
```
*mercenary_set_calls <guild>,<value>;
```

Sets or gets the mercenary calls value for given guild for currently
attached character. Guild can be one or the following constants:

	ARCH_MERC_GUILD
	SPEAR_MERC_GUILD
	SWORD_MERC_GUILD
