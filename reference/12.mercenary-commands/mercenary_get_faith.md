### mercenary_get_faith
```
*mercenary_get_faith(<guild>);
```
### mercenary_set_faith
```
*mercenary_set_faith <guild>,<value>;
```

Sets or gets the mercenary faith value for given guild for currently
attached character. Guild can be one or the following constants:

	ARCH_MERC_GUILD
	SPEAR_MERC_GUILD
	SWORD_MERC_GUILD
